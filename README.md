# 🔊 Text To Speech Converter

A simple and responsive Text to Speech Converter web application built using HTML, CSS, and JavaScript.  
This application converts user-entered text into spoken voice using the Web Speech API.

---

## 🚀 Features

- Convert text into speech instantly
- Clean and user-friendly interface
- Supports multiple voice options (based on browser support)
- Fully responsive design
- Works directly in the browser (No backend required)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Speech API

---

Text-to-Speech-Converter/

---

## 🎯 How It Works

1. User enters text into the textarea.
2. When the "Speak" button is clicked:
   - JavaScript captures the text.
   - Web Speech API converts text into speech.
   - The browser plays the generated voice.

---

## 💻 How to Run the Project

1. Download or clone the repository:

git clone https://github.com/Shivvannisinha/Text-to-Speech-Converter.git


2. Open the project folder.
3. Open `index.html` in your browser.
4. Start typing and click the **Speak** button.
